#### 🏆 [desafio_samu_hackathon](https://github.com/olvvikitor/desafio_samu_hackathon)
> Sistema de atendimento de emergência com SMS e geolocalização — **3º lugar em hackathon**

- Envio de **SMS automático** via Twilio ao solicitante com link de localização
- **Geolocalização em tempo real** via navegador vinculada ao chamado
- Dados de localização e incidente enviados diretamente ao **motorista da ambulância**
- Arquivo **Insomnia** incluso para testes da API
- Resolve problemas reais: trotes, localização imprecisa e falta de vagas em UBS

`TypeScript` `NestJS` `Twilio SMS` `Geolocalização` `MongoDB`
